PHPの変数の特徴：

  • $変数名 の形式で宣言
  • 事前宣言不要（いきなり使用可能）
  • 変数名は英数字とアンダースコア（_）のみ使用可能
変数の表示（var_dump / echo）
  • 変数の中身を確認するには、var_dump() と echo の2つの方法が存在する

var_dump() - デバッグ用

  • 変数の 値と型（データ型） を両方表示する
<?php
$name = "どんぶラッコ";
$birthMonth = 1;
$isStudent = true;

var_dump($name);      // string(18) "どんぶラッコ"
var_dump($age);       // int(29)
var_dump($isStudent); // bool(true)
?>

echo - 出力用

  • 値をそのまま文字列として表示する
  • HTMLへの出力に使う
<?php
$name = "どんぶラッコ";
$birthMonth = 1;

echo $name;           // どんぶラッコ
echo $birthMonth;            // 1
?>
var_dumpとechoの使い分け

  • var_dump(): 変数の中身をデバッグしたい時（型も確認できる）

  • echo: 画面に値を表示したい時（HTMLへの出力）

  • 開発中は var_dump() で変数の状態を確認し、完成したら echo で表示するのが一般的

文字列の結合

基本的な結合（ドット演算子）：

<?php
$name = "どんぶラッコ";
echo "私の名前は" . $name . "です";
?>

ダブルクォーテーションを使った変数展開：

<?php
$name = "どんぶラッコ";
echo "私の名前は $name です";           // 変数が展開される
echo "私の名前は{$name}です";         // より明確な記法
?>

シングルクォーテーションの場合：

<?php
$name = "どんぶラッコ";
echo '私の名前は $name です';         // $nameがそのまま表示される
?>
クォーテーションの使い分け
  • ダブルクォーテーション（"）: 変数が展開される
  • シングルクォーテーション（'）: 変数は展開されず、文字列として扱われる

処理速度はシングルクォーテーションの方がわずかに高速ですが、実用上の差はほとんどありません。

結合演算子（.=）：

<?php
$name = "どんぶラッコ";
$message = "私の名前は";
$message .= $name;           // JavaScriptの += に相当
$message .= "です";
echo $message;
?>

XSS（クロスサイトスクリプティング）対策

XSSの危険性を体験：

フォームのactionをpost_confirm_xss.phpに変更してください：

<form action="post_confirm_xss.php" method="post">
    お名前: <input type="text" name="name">
    Email: <input type="text" name="email">
    <input type="submit" value="送信">
</form>

以下のHTMLタグやJavaScriptをフォームに入力してみましょう：

<h1>見出しの送信</h1>
<script>alert("XSS攻撃成功！")</script>

問題が発生する理由：

  • ユーザーの入力をそのままHTMLとして出力すると、HTMLタグやスクリプトが実行される
  • 悪意あるスクリプトが実行される可能性がある

対策：htmlspecialchars()関数

post_confirm_xss.phpでは、htmlspecialchars()関数を使ってユーザー入力をエスケープしています：

<?php
$name = $_POST["name"];
$email = $_POST["email"];
?>

<li>お名前: <?= htmlspecialchars($name, ENT_QUOTES, 'UTF-8') ?></li>
<li>メール: <?= htmlspecialchars($email, ENT_QUOTES, 'UTF-8') ?></li>

htmlspecialchars()の役割：

  • HTMLの特殊文字をエスケープして、安全に表示する
  • 第1引数：エスケープする文字列
  • 第2引数：ENT_QUOTES - シングルクォートとダブルクォートの両方をエスケープ
  • 第3引数：'UTF-8' - 文字エンコーディングを指定

エスケープされる文字：

  • < → &lt; （HTMLタグの開始記号）
  • > → &gt; （HTMLタグの終了記号）
  • " → &quot; （ダブルクォート）
  • ' → &#039; （シングルクォート、ENT_QUOTES使用時）

これにより、<script>alert("XSS")</script>のような入力も、そのままテキストとして表示され、実行されなくなります。

セキュリティは最重要

ユーザーからの入力は必ずhtmlspecialchars()でエスケープしましょう！

データベースに保存する前ではなく、HTMLに出力する直前でエスケープすることが重要です。

ファイルへの書き込み

基本的なファイル書き込み：

<?php
// 現在の日時を文字列として作成
$currentTime = date("Y-m-d H:i:s");

// ファイルを開く（'a'は追記モード）
$file = fopen("data/data.txt", "a");

// ファイルに書き込み
fwrite($file, $currentTime . "\n");

// ファイルを閉じる
fclose($file);

echo "データを書き込みました！";
?>

ファイルモードの種類：

モード説明用途
"r"読み取り専用ファイルの内容を読む
"w"書き込み専用（上書き）ファイルを新規作成・上書き
"a"書き込み専用（追記）ファイルの末尾に追加
"r+"読み書き既存ファイルの読み書き
フォームデータをファイルに保存

フォームからのデータ書き込み：

<?php
// write.php
if ($_POST) {
    $name = $_POST["name"];
    $email = $_POST["email"];
    $timestamp = date("Y-m-d H:i:s");
    
    // CSVフォーマットでデータを作成
    $csvData = $timestamp . "," . $name . "," . $email . "\n";
    
    // ファイルに書き込み
    $file = fopen("data/data.csv", "a");
    fwrite($file, $csvData);
    fclose($file);
    
    echo "データを保存しました！";
}
?>

CSVフォーマット

CSV（Comma Separated Values）は、データをカンマで区切って保存する形式です：

2025-01-15 14:30:25,田中太郎,tanaka@example.com 2025-01-15 14:35:12,佐藤花子,sato@example.com

ExcelやGoogleスプレッドシートでも開ける汎用的な形式です。

ファイルからの読み込み

ファイル内容の表示：

<?php
// read.php
echo "<h1>登録データ一覧</h1>";

// ファイルが存在するかチェック
if (file_exists("data/data.csv")) {
    $file = fopen("data/data.csv", "r");
    
    echo "<table border='1'>";
    echo "<tr><th>登録日時</th><th>お名前</th><th>Email</th></tr>";
    
    // ファイルを1行ずつ読み込み
    while ($line = fgets($file)) {
        $line = trim($line);  // 改行文字を除去
        if ($line !== "") {
            $data = explode(",", $line);  // カンマで分割

設定ファイルの読み込み：

<?php
// config.txt の内容
// site_name=マイサイト
// max_upload_size=5MB
// admin_email=admin@example.com

function loadConfig($filename) {
    $config = [];
    $file = fopen($filename, "r");
    
    while ($line = fgets($file)) {
        $line = trim($line);
        if ($line && strpos($line, '=') !== false) {
            list($key, $value) = explode('=', $line, 2);
            $config[trim($key)] = trim($value);
        }
    }
    
    fclose($file);
    return $config;
}

$config = loadConfig("config.txt");
echo $config['site_name'];  // "マイサイト"
?>
